      Validation Rules

      #isRequired()

      Test if there is value.

      import { isRequired } from '@formiz/validations'
      <MyField
        name="fieldName"
        validations={[
          {
            rule: isRequired(),
            message: 'This field is required',
          },
        ]}
      >

      #isNotEmptyString()

      Test if a string has not only spaces.

      import { isNotEmptyString } from '@formiz/validations'
      <MyField
        name="fieldName"
        validations={[
          {
            rule: isNotEmptyString(),
            message: 'This field can\'t be empty',
          },
        ]}
      >
