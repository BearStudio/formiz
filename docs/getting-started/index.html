      Getting Started

      @formiz/core is in alpha, API is subject to change. Do no use in production.

      #Install Formiz core

      With Yarn

      yarn add @formiz/core

      With NPM

      npm install @formiz/core

      #Create your first custom field

      A custom field can be anything, not just inputs.

      Here there is an example with an input, but you can litteraly turn anything into a field and you will get validation for free!

      import React from 'react'
      import { useField } from '@formiz/core'
      export const MyField = (props) => {
        const { value, setValue, errorMessage, isValid, resetKey } = useField(props)
        const { label } = props
        return (
          <div>
            <label>
              { label }
            </label>
            <input key={resetKey} type="text" defaultValue={value} onChange={(e) => setValue(e.target.value)} />
            {!isValid && (
              <p>
                { errorMessage }
              </p>
            )}
          </div>
        )
      }

      #Use your field in a form

      Then use your field as many time as you want :)

      Each field need a property "name" which will be the key in the object of values.
      So, be sure that this key is unique by form.

      import React from 'react'
      import { Formiz, useForm } from '@formiz/core'
      import { MyField } from './MyField'
      export const MyForm = () => {
        const myForm = useForm()
        const submitForm = (values) => {
          console.log(values) // { firstName: 'value', lastName: 'value' }
        }
        return (
          <Formiz onValidSubmit={submitForm} connect={myForm}>
            <form onSubmit={myForm.submit}>
              <MyField
                name="firstName"
                label="First Name"
                isRequired="First Name is required"
              />
              <MyField
                name="lastName"
                label="Last Name"
                isRequired="Last Name is required"
              />
              <button type="submit" disabled={!myForm.isValid}>
                Submit
              </button>
            </form>
          </Formiz>
        )
      }

      #ðŸ§™ Multi Steps form

      You already have your field so we just need to update our form to use steps.

      Import the FormizStep component and use it to split the form into steps.

      We need also need to update the submit button to allow the user to navigate between steps.

      import React from 'react'
      import { Formiz, FormizStep, useForm } from '@formiz/core'
      import { MyField } from './MyField'
      export const MyForm = () => {
        const myForm = useForm()
        const submitForm = (values) => {
          console.log(values) // { firstName: 'value', lastName: 'value' }
        }
        return (
          <Formiz onValidSubmit={submitForm} connect={myForm}>
            <form onSubmit={myForm.submit}>
              <FormizStep name="step1">
                <MyField
                  name="firstName"
                  label="First Name"
                  isRequired="First Name is required"
                />
              </FormizStep>
              <FormizStep name="step2">
                <MyField
                  name="lastName"
                  label="Last Name"
                  isRequired="Last Name is required"
                />
              </FormizStep>
              {!myForm.isFirstStep && (
                <button type="button" onClick={myForm.prevStep}>
                  Back
                </button>
              )}
              {myForm.isLastStep ? (
                <button type="submit" disabled={!myForm.isValid}>
                  Submit
                </button>
              ) : (
                <button type="button" onClick={myForm.submitStep} disabled={!myForm.isStepValid}>
                  Continue
                </button>
              )}
            </form>
          </Formiz>
        )
      }
