# Formiz

This component creates the form context.

## Import

```tsx
import { Formiz } from "@formiz/core";
```

## Props

### connect

<Type type="Form" />
Connect your form context with the form (from `useForm()` hook).

See [`useForm()`](/core/useForm) documentation for more details.

```tsx
const MyForm = () => {
  const form = useForm();

  return <Formiz connect={form}>
    {/* Your fields here */}
  </Formiz>;
};
```

### autoForm

<Type type="boolean | 'form' | 'step'"/>
- Set to `true` or `'form'` to wrap children in a `<form>` element with auto `submit()` on form submission.
- Set to `'step'` to wrap children in a `<form>` element with auto `submitStep()` on form submission.

```tsx
const form = useForm();

<Formiz connect={form} autoForm>
  {/* your fields here */}
  <button type="submit">Submit</button>
</Formiz>;
```
